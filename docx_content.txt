[Heading 1] 1. Por que Python? O que é e onde é usado
[Normal] Python é, hoje, a linguagem de programação mais popular do mundo para automação, dados e inteligência artificial. Criada por Guido van Rossum em 1991, ela foi projetada para ser legível como inglês — o que a torna ideal para quem está começando e para quem precisa de produtividade no dia a dia.
[Heading 2] 1.1 Vantagens do Python
[List Paragraph] Legibilidade: o código se parece com inglês, facilitando leitura e manutenção
[List Paragraph] Ecossistema enorme: bibliotecas prontas para automação, dados, IA, web e muito mais
[List Paragraph] Alta demanda: linguagem mais pedida em vagas de automação e análise de dados no Brasil
[List Paragraph] Multiplataforma: roda em Windows, Mac e Linux sem alteração no código
[List Paragraph] Comunidade ativa: fácil encontrar soluções, tutoriais e suporte online

[Heading 1] 2. Variáveis e Tipos de Dados
[Normal] Variáveis são 'caixinhas' que guardam informações na memória do computador. Ao contrário de outras linguagens, em Python você não precisa declarar o tipo — ele é detectado automaticamente.
[Heading 2] 2.1 Tipo str — Texto
[Normal] Strings armazenam qualquer sequência de caracteres. Use para nomes, mensagens, caminhos de arquivo, CPFs, e tudo que não é número para cálculo.
[Heading 2] 2.2 Tipo int — Número Inteiro
[Normal] Inteiros são números sem parte decimal. Use para contagens, IDs, índices, quantidade de itens, anos.
[Heading 2] 2.3 Tipo float — Número Decimal
[Normal] Floats armazenam números com casas decimais. Use para valores monetários, porcentagens, medições. Atenção: ponto como separador decimal (não vírgula).
[Heading 2] 2.4 Tipo bool — Verdadeiro/Falso
[Normal] Booleanos só aceitam True ou False (com maiúscula). São a base de toda lógica condicional.
[Heading 2] 2.5 Verificando tipos com type()
[Heading 2] 2.6 f-strings — Formatação moderna
[Normal] f-strings são o jeito mais moderno e legível de montar textos com variáveis em Python. Coloque um f antes das aspas e use {} para inserir qualquer variável ou expressão.

[Heading 1] 11. O que é Automação de Processos (RPA)
[Normal] RPA (Robotic Process Automation) é a técnica de usar software para automatizar tarefas repetitivas que humanos fazem manualmente em computadores — clicar em botões, preencher formulários, copiar dados entre sistemas.
[Heading 2] 11.1 Onde RPA se aplica
[List Paragraph] Extrair dados de sistemas legados sem API disponível
[List Paragraph] Preencher formulários web ou desktop automaticamente
[List Paragraph] Copiar informações de um sistema para outro (ERP → planilha, por ex.)
[List Paragraph] Gerar relatórios periódicos sem intervenção manual
[List Paragraph] Enviar e-mails em massa com dados personalizados
[List Paragraph] Monitorar sites e alertar sobre mudanças de preço ou estoque
